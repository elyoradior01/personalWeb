Movie Theatre System Application in C# and SQL Database


   As part of my Application Development course's final project, we were tasked with creating a cinema/movie theater system using C# programming language and a SQL Database. The project was divided into two main sections: a ticket booking system and a food and beverages ordering system, each handled by a team of two.
   Our ticket booking system incorporated several essential features, including seat selection, various cinema types (ranging from regular to premier), booking details, and database integration for sales tracking. Similarly, our snack ordering system featured all the necessary cashier ordering essentials, with its data also being stored in the database for efficient management.
   Although each team member contributed equally, I specifically took on the responsibility of developing the database for the various cinema types, a portion of the design process, as well as coding the majority of the snack ordering cashier system.
